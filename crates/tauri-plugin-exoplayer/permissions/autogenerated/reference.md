## Default Permission

ExoPlayer playback commands for Android.

#### This default permission set includes the following:

- `allow-load`
- `allow-play`
- `allow-pause`
- `allow-stop`
- `allow-setVolume`
- `allow-setMuted`
- `allow-showPlayer`
- `allow-hidePlayer`
- `allow-getStats`
